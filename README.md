# CHEREPOVETS Training

Тренажёр Discord-модерации для состава CHEREPOVETS.

## Что внутри

- тёмный интерфейс в стиле штаба;
- авторизация и кабинет модератора;
- тренажёр кейсов: факт, правило, доказательства, действие;
- проверки решений и итоговые задания;
- seed с демо-пользователем и модераторскими кейсами.

## Команды

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

Демо-вход после seed:

```text
student@example.com
password123
```
