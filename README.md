# Cherepovets 89 — clean full no-organizations

Сборка из текущего `index.html` с переносом в нормальную структуру статического сайта.

Структура:
- `index.html` — разметка и подключения файлов;
- `assets/css/app.css` — стили сайта + мобильная адаптация;
- `assets/js/app.js` — основной код сайта;
- `assets/img/favicon.png` — favicon, вынесенный из base64.

Что убрано:
- верхний раздел «Организации»;
- поднавигация организаций;
- видимые разделы ФСБ/ГОСС/организаций;
- отдельные runtime-модули организаций;
- Cloudflare beacon-дубли;
- мои предыдущие safe/stable runtime-патчи.
