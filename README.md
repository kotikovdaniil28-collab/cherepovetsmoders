# CHEREPOVETS Moderation v56

## v56: дашборд сразу на главной

- Главная страница `/` теперь сразу открывает настоящий личный дашборд `/native/profile`.
- `/native` больше не показывает промежуточные плитки и тоже ведёт прямо в дашборд.
- Старый экран авторизации сохранён отдельно на `/login?legacy=1`.
- После входа достаточно открыть корень сайта — старый интерфейс больше не перекрывает дашборд.

## v55: видимый дашборд и экспорт активности

- В старом интерфейсе появилась заметная кнопка `Открыть новый дашборд`.
- `/profile`, `/reports`, `/inactives`, `/reports-table` и `/leaderboard` ведут на обновлённые страницы; старый интерфейс доступен через `?legacy=1` и кнопку `Старый сайт`.
- Таблица активности получила недельную сводку: выполнение, одобренные отчёты, сильные результаты и пропуски/отказы.
- Добавлен экспорт PNG для публикации в INFO и CSV для Excel.

## v54: карьерный дашборд и VK-вердикты

- `/native/profile` стал полноценным личным кабинетом модератора.
- Показываются реальная должность из `Discord состав`, дни на ступени, отчёты, XP и два пути повышения.
- `/native/review` получил красивое меню решения вместо ряда технических кнопок.
- Руководитель может изменить заявленный тип отчёта или выбрать готовую причину отказа.
- Решение проходит через защищённый server route, обновляет `reports`, пишет аудит и отправляет вердикт в VK ЛС.
- Если отчёт пришёл из VK, карточка с кнопками в группе также обновляется после решения на сайте.
- Native-таблица использует тот же маршрут решений, поэтому вердикты из неё тоже приходят модератору.
- Главная native-панель, навигация и профиль получили обновлённый адаптивный дизайн.

Перед деплоем выполните SQL из архива бота и добавьте сайту серверные переменные:

```env
SUPABASE_URL=https://YOUR_PROJECT.supabase.co
SUPABASE_ANON_KEY=sb_publishable_YOUR_KEY
SUPABASE_SERVICE_ROLE_KEY=sb_secret_YOUR_SERVICE_ROLE_KEY
VK_GROUP_TOKEN=vk1.a.YOUR_GROUP_TOKEN
VK_API_VERSION=5.199
```

`SUPABASE_SERVICE_ROLE_KEY` и `VK_GROUP_TOKEN` используются только server route и не попадают в браузерный JavaScript.
